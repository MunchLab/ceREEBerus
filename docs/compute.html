Code started by Liz Nov 2022. The goal is to get a merge tree class with the following properties. - Should accept a Reeb graph class as input. - Needs a check to make sure the input is actually a Reeb graph. In particular, only down forks and a root node with function value np.inf. - Has drawing capabilities, in particular can handle that np.inf node. - If the Reeb graph passed in isn’t a merge tree, likely I just want to generate the merge tree of the input Reeb graph.
