Interleave class
================

This is the ``Interleave`` class which is used to construct an interleaving.  

Note that this class can be imported as::

    from cereeberus import Interleave
    myInt = Interleave(M1, M2, n)

A full tutorial can be found in the `Interleaving Basics Jupyter Notebook <../../notebooks/interleaving_basics.ipynb>`_.
