1. Examples of Reeb graphsÔÉÅ

In the following notebook, we show examples of Reeb graphs that are included in the package which show functionality related to the main class. All Reeb graphs are stored as the class cereberus.reeb.Reeb.

[2]:

reeb_torus = ex_reebgraphs.torus()
reeb_torus.draw()

The Reeb Graph class has many attributes available such as nodes, edges, position, up degree, down degree, and function values provided at class instantiation

[3]:

print("Reeb Graph nodes: ", reeb_torus.nodes)
print("Reeb Graph edges: ", reeb_torus.edges)
print("Reeb Graph function values: ", reeb_torus.f)
# print("Reeb Graph up degree: ", reeb_torus.up_deg)

#print("Reeb Graph Adjacency Matrix:", reeb_torus.adjacency)

There is also a custom plot function to plot Reeb Graphs which can be called with plot_reeb.

[4]:

reeb_torus.draw()

[5]:

reeb_torus.draw_straight_line(with_labels=True)

You can also reset the positions completely by using the set_pos_from_f method. If you pass a seed, it will make the resulting positions the same every time.

[6]:

reeb_torus.set_pos_from_f(seed = 12)
reeb_torus.draw()

We can also remove all regular nodes (i.e. those with both up- and down-degree equal to 1) to get an equivalent, minimal Reeb graph. Note that we‚Äôre using the draw command here since draw_straight_line will make the multiedges land on top of each other and so not be visible.

[22]:

R.draw(cpx = 15)
