2. Reeb Graph

This is the reeb graph class which is the majority of functionality of the package. This includes funtionality such as assigning function values to nodes in the Reeb graph and plotting the graph.

class cereeberus.reeb.graph.Reeb(G, fx={}, horizontalDrawing=False, verbose=False)[source]

Class for Reeb Graph

Variables

  • G – Graph: G

  • fx – function values associated with G

  • pos – spring layout position calculated from G

  • pos_fx – position values corresponding to x = fx and y = y value from pos

plot_reeb(position={}, resetSpring=False, horizontalDrawing=False, verbose=False, cpx=0.1, cpy=0.1)[source]

Plot a Reeb Graph given a graph with a position. If no position passed, it will take the spring layout version. In this case, it will either be drawn vertically or horizontally, depending on the horizontalDrawing (boolean) passed in.
