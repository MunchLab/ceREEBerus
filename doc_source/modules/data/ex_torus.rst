Torus Lower Star Simplicial Complex
*************************************

This module contains the `Torus` class, which generates a simplicial complex representing a torus as an nxn grid, and gives it a lower star filtration. 
Pictures of these can be seen in `this notebook <../../notebooks/compute_Reeb_from_simplex_tree.html>`_
